# Конспект по Git

## Описание команд

### Проверка работы Git
    git --version

### Представляемся Git
    git config --global user.name "Name"
    git config --global user.email "Email Address"

### Создаем репозиторий
    git init

### Добавляем файл/файлы для отслеживания
    git add - все
    git add "file_name"

### Сохраняем изменения
    git commit -m "text"

### Журнал изменений
    git log

### Переключение между версиями
    git checkout <название версии>
